Cybersecurity Tools

مجموعة من أدوات الأمن السيبراني القوية لاختبار الاختراق، وفحص الثغرات، وتقييم الأمن.

A collection of powerful cybersecurity tools for penetration testing, vulnerability scanning, and security assessment.

XSStrike

أداة متقدمة للكشف عن ثغرات XSS واستغلالها.

Advanced XSS detection and exploitation tool.

Kali Commands

أوامر كالي لينكس الأساسية لاختبار الاختراق.

Essential Kali Linux commands for penetration testing.

Katana

إطار عمل متطور للزحف واستكشاف المواقع.

A next-generation crawling and spidering framework.

Nikto

ماسح خوادم الويب للملفات الخطرة والثغرات الأمنية.

Web server scanner for dangerous files and vulnerabilities.

Nuclei

ماسح ثغرات سريع وقابل للتخصيص.

Fast and customizable vulnerability scanner.

Subfinder

أداة اكتشاف النطاقات الفرعية لتقييمات الأمان.

Subdomain discovery tool for security assessments.

Wapiti

ماسح ثغرات تطبيقات الويب.

Web application vulnerability scanner.

XSSer

إطار عمل آلي للكشف عن ثغرات XSS.

Automated framework for XSS vulnerability detection.

SQLMap
